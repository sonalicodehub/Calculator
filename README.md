# 📱 Calculator 

A clean and simple **Calculator Application** built using **Python** and **Tkinter**. This desktop calculator provides basic arithmetic operations with a user-friendly graphical interface, making it ideal for beginners learning Python GUI development.

---

## ✨ Features

➕ Basic arithmetic operations (+, −, ×, ÷)
🔢 Decimal number support
🧹 Clear button to reset calculations
🖥 Simple and intuitive GUI
⚡ Instant result calculation
🐍 Built entirely with Python and Tkinter
🎓 Beginner-friendly project

---

## 🎮 Calculator Controls

| Button | Action           |
| ------ | ---------------- |
| 0–9    | Number input     |
| +      | Addition         |
| -      | Subtraction      |
| *      | Multiplication   |
| /      | Division         |
| .      | Decimal point    |
| =      | Calculate result |
| Clear  | Clear all input  |

## 🤖 How the App Works

* Users enter numbers and operators by clicking calculator buttons.
* Input is displayed in the text field at the top.
* Mathematical expressions are created dynamically.
* When **=** is pressed, the expression is evaluated.
* The result is displayed instantly.
* Invalid expressions show **"Error"**.
* The **Clear** button removes all current input.
